开始实习

2014-07-05
生活
约 1038 字 预计阅读 3 分钟

【注意】最后更新于 2014-07-05, ，文中内容可能已过时，请谨慎使用。

艰难无比令人欲生欲死的考试终于过去了，如果不出意外的话，应该是不会挂了吧！当然，现在说这些话还早，明天才出成绩，等着吧。

由于地球人都会选择大三暑假去实习，作为一个正常的呆萌地球人，我肯定是义不容辞的冲在实习的最前线。这不，刚考完试，我就屁颠屁颠跑去实习了。

之前找到的实习单位有俩个，一个公司比较大，有几百人吧。然后主要搞医疗的，我如果进去是搞H5的app开发，算是附属产品吧。那里有点偏，公司研发部又有点破，入职又麻烦，还要签一年。由于以上种种原因，我选择去一家很小的创业公司。

这家公司其实一开始不打算去的，虽然工资不低，做的也是主要业务，但是！要自己背电脑去！这对于一个对背电脑深恶痛绝的人来说，是绝对不能接受的！所以我考虑不久就说不考虑了。后来，后来，后来因为没有找到其他更好的，这公司又同意加点工资，我就毫不犹豫的扑上去了。这不，上班都俩天了。

这俩天上班基本没干啥，都是熟悉环境，看规范，看源码。

第一天，自我介绍，搭环境，看源码，学习新技术API，就这样过去了。

第二天，看源码，看API，还是这样过去。

来总结下这俩天的情况吧。主动做人对于我还是有点难，我还是一如既往的沉默是金。在别人眼里应该是不太活跃吧。但来到新的环境，对周围的一切都让我感到很陌生，以至于我有点小心翼翼。这也许是我保守的地方。我很难对大部分人敞开心扉，或者说不熟悉的人。这不知道是好还是不好。然后做事还是不大认真，对技术没有那种如饥似渴的感觉。有种应付式的感觉，我知道这是很危险的信号。因为不懂的太多，有点定不下心来研究一个问题。这无论是对工作还是以后个人的发展都是很不利的。还是那句话：要做就尽力去做，不要留下遗憾。

然后说好玩的事情吧，由于公司主要分研发和推广，每天早上，看到推广人员来一下公司，然后就出去找客户，然后，一天就不见人了。哈哈，我一直觉得这样的模式很好玩。老总是管推广的，每天他也来一下，然后就不见人影了。让我觉得好神奇。

对于这个公司，不管以后发展怎样，能不能拿到风投，能不能做大，我觉得都不会后悔。一是因为公司用的技术都是我接触过但没有实质操作过的，所以技术方面可以得到提升。二是技术方面的负责人特别有心，只要你问一个问题，他会举一反三的告诉你原因。我刚去第一天就把整个项目的框架，用到的技术都给我梳理了一遍。这让我觉得很难得，真的挺难得的。而且他还会抽时间看我们的代码，把缺点说出来。在我眼里，一个好的技术团队也不过如此了。

恩，得回去洗白白了。图书馆的电脑也太卡了，卡的令人发指！
